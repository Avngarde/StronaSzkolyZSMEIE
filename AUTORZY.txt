# Strona ZESPOŁU SZKÓŁ MECHANICZNYCH ELEKTRYCZNYCH I ELEKTONICZNYCH

Jest to projekt strony internetowej dla Zespołu Szkół Mechanicznych Elektrycznych i Elektronicznych w Toruniu.

## Użyte technologie:

### Frontend

Wykorzystaliśmy standardowo:

 * HTML
 * CSS
 * JavaScript
 
A także:
    * Google Maps
    * Google Fonts 
	  * Ropa Sans
	  * Audiowide

### Backend

Użyte przez nas technologie backendowe to:

  * Django - Framework napisany w języku Python, łatwy i wygodny, pozwalający w nie męczący sposób projektować naszą aplikacje internetową.

  * Python - Język programowania w którym został napisany Django, to w nim napisaliśmy większość skryptów dla strony ZSMEiE.

  * PostgreSQL - Wykorzystaną przez nas technologią baz danych jest PostgreSQL.
